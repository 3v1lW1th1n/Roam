IntraMaps Roam

Simple and flexible data collection built using QGIS.

Download Zip Download Installer

IntraMaps Roam is a field, and office, data collection application built using the QGIS libraries and Python. It was built to be easy to use on (windows) based devices in an offline situation without a internet connection and the need for easy to use data collection. Roam supports point, line, and polygon capture, include Point at GPS capture.

Roam supports custom Qt forms, or the much simpler but much more rapid, auto generated forms. Both form types support the same widget types and allow for custom Python logic to be added in order to fit the exact workflow needed.

  • Info2 and Info1 block support

  • Better link support in info panel

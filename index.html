IntraMaps Roam is a field data collection application built using the QGIS libraries and Python. It was built to be easy to use on (windows) based tablet devices, with Android planned for the future.

See the wiki for information on how to create a project and user guides.

The latest release can be found on the release page

Newest features in 2.1 include:

  • Poylgons and lines
  • Drawing pad
  • Camera (Windows only)
  • GPS page
  • Translation support
  • API clean up and extras (access to GPS on forms etc)
  • Legend page
  • Linux support
  • Attribute Help

More details can be found on the features page for the release.
